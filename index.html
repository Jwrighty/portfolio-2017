NihonGo

NihonGo! (meaning Japanese in...Japanese) is an app for learning the Japanese Hiragana and Katakana character sets. Through my own experiences of trying to learn what is a pretty complicated writing system I developed an app that allows the learning and practice of character sets, measuring progress through tests and moving on to learning to construct words.
Built with a Rails API paired with a PostgresSQL database and AngularJS front end the app is fully authenticated.
